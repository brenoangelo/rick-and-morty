# Rick and morty

## Introdução

Esse projeto foi criado com intuito de aprender as novas features do React 18.


## Tecnologias

- ReactJS
- Typescript
- Vite
- TailwindCss
- rick and morty api

## Como rodar ?


### Instalando dependencias

```bash
  yarn
```

### Rodando o projeto

```bash
  yarn start
```
